# Dataset: учебные видео и разметка

Набор данных подготовлен для дипломного прототипа системы анализа упражнений по пауэрлифтингу.

Состав:

- 10 видеороликов в `sample_videos`;
- 10 CSV-файлов с координатами позы в `sample_pose`;
- `dataset/annotations/video_labels.csv` — ручная разметка упражнения, качества и типа ошибки;
- `dataset/annotations/program_results.csv` — результат запуска программы на этих роликах;
- `dataset/annotations/rep_annotations.csv` — разметка найденных повторений;
- `dataset/annotations/pose_index.csv` — соответствие видео и CSV с позой.

Классы упражнений:

- `squat` — присед;
- `bench` — жим лежа;
- `deadlift` — становая тяга.

Метки качества:

- `good` — хороший учебный пример;
- `bad` — пример с ошибкой.
